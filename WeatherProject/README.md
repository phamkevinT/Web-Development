# Weather App.

A basic web-application that uses the OpenWeatherMap API and user input to display the current temperature and current weather condition of a city.

## Installation

Download the WeatherProject folder and run the index.html in a web browser.



## Topics Covered

```python
- API Endpoints, Paths, Parameters
- API Authentication & Postman (App)
- Introduction to JSON
- GET Requests using Node HTTPS Module
- Parsing JSON
- Express to render website using Live API Data
- Body Parser to Parse POST Requests to a Server
```
